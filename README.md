# Cpp Console Projects

A collection of small C++ console applications for practice and learning.

## Projects

- **Project_01_RockPaperScissor.cpp** — Rock/Paper/Scissors game against the computer with multiple rounds.
- **Project_02_MathGame.cpp** — Arithmetic quiz with difficulty levels and operation types.
- **Project_03_Bank.cpp** — Basic bank system for listing, adding, deleting, updating, and finding clients.
- **Project_04_BankExtension1.cpp** — Bank system extension with deposits, withdrawals, and total balances.
- **Project_05_BankExtension2.cpp** — Bank system extension with login, user management, and permissions.
- **Project_06_ATMSystem.cpp** — ATM interface with login, quick/normal withdrawals, deposits, and balance checks.

## Data Files

Some projects read/write local text files in the repo root:

- **ClientsData.txt** — Used by the bank and ATM projects.  
  Format: `AccountNumber#//#PinCode#//#Name#//#Phone#//#Balance`
- **UsersData.txt** — Used by the BankExtension2 project.  
  Format: `Username#//#Password#//#Permissions`

Sample records are included in both files for quick testing.
