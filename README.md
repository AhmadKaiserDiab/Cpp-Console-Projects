# Cpp Console Projects

A collection of small C++ console applications for practices and learning and Implementing different Programming Principles
## Projects
- **Project 01: RockPaperScissor** — Rock/Paper/Scissors game against the computer with multiple rounds.
- **Project 02: MathGame** — Arithmetic quiz with difficulty levels and operation types.
- **Project 03: Bank** — Basic bank system for listing, adding, deleting, updating, and finding clients.
- **Project 04: BankExtension1** — Bank system extension 1 with deposits, withdrawals, and total balances.
- **Project 05: BankExtension2** — Bank system extension 2 with login, user management, and permissions.
- **Project 06: ATMSystem** — ATM interface with login, quick/normal withdrawals, deposits, and balance checks.

## Data Files
Some projects read/write local text files in the repo root:

- **ClientsData.txt** — Used by the bank and ATM projects.  
  Format: `AccountNumber#//#PinCode#//#Name#//#Phone#//#Balance`
- **UsersData.txt** — Used by the BankExtension2 project.  
  Format: `Username#//#Password#//#Permissions`

> Sample records are included in both files for quick testing.
